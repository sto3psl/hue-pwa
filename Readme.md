# hue

Small web application to control Philips Hue light bulbs.

## Setup

Create a `config.json` file in `/src` with your credentials.

```json
{
  "userId": "{userId}",
  "url": "{url}"
}

```

Install dependencies.

```bash
$ npm i
```

## Development

Start the dev server and get cracking.

```bash
$ npm start
```

## Serve the application

```bash
$ npm run serve
```
